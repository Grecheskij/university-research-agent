# Agent Core Rules

- Все инструменты должны возвращать единый формат `Paper` из `data/schemas/paper_schema.py`.
- Агент строится по паттерну ReAct поверх `LangChain`.
- Агент обязан планировать шаги, вызывать инструменты и агрегировать результаты.
- Для RAG используется `ChromaDB`, коллекция `research_papers`.
- Системные промпты и служебные инструкции хранятся на русском языке.
- Агент должен корректно обрабатывать запросы на русском и английском языках.
- Агент должен определять язык запроса и отвечать на том же языке, если пользователь не просит иной.
- При необходимости допустим внутренний перевод через LLM без показа промежуточных шагов пользователю.
- На уровне агрегирующих цепочек нельзя возвращать более 10 статей за один поиск.
- Форматы цитирования, которые должны поддерживаться позже: `APA 7`, `MLA 9`, `ГОСТ 7.0.5`.
- Ответы должны содержать ссылки на источники и DOI там, где это возможно.
