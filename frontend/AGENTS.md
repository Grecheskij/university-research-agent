# Frontend Agent Rules

- Интерфейс строится на `Gradio`.
- UI должен содержать вкладки: `Поиск`, `Обзор`, `Библиография`, `Аналитика`.
- Интерфейс обязан поддерживать переключатель языка `RU/EN`.
- Все подписи, названия вкладок, кнопок и подсказки должны иметь локализации для русского и английского языков.
- Пользовательский ввод может быть как на русском, так и на английском.
- Результаты должны поддерживать Markdown и отображать ссылки на источники рядом с ответами.
- Фронтенд общается только с backend через задокументированные FastAPI-эндпоинты.
- Прямой доступ UI к `agent_core`, ChromaDB или другим данным запрещён.
