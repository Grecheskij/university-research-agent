# Backend Agent Rules

- Backend строится на `FastAPI` с `async def` роутами.
- Валидация входных и выходных данных выполняется через `Pydantic v2`.
- Backend является единой точкой входа для frontend.
- UI не должен обращаться к `agent_core` и данным напрямую.
- Роуты разделяются по функционалу: поиск, обзор, библиография, аналитика.
- CORS должен разрешать только localhost для разработки и домен Hugging Face Space для продакшена.
- Ошибки должны возвращаться в структурированном формате с кодом и сообщением.
- Нужно предусмотреть ограничение размера тела запроса и базовый rate limiting по мере реализации.
