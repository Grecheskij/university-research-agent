# DevOps Rules

- `requirements.txt` должен содержать все зависимости проекта с зафиксированными версиями.
- `.env.example` должен перечислять все переменные окружения без реальных значений.
- README должен быть на русском языке.
- В README должны быть инструкции локального запуска для backend и frontend.
- В README должен быть раздел по деплою на Hugging Face Spaces.
- В README должен быть раздел по безопасной работе с секретами через Hugging Face Secrets и GitHub Secrets.
- Связь между GitHub-репозиторием и Hugging Face Space должна быть задокументирована.
