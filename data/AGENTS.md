# Data Agent Rules

- Коллекция `ChromaDB` должна называться `research_papers`.
- Эмбеддинги позже настраиваются через `sentence-transformers` с моделью `all-MiniLM-L6-v2`.
- Перед сохранением требуется дедупликация по DOI.
- Если DOI отсутствует, дедупликация выполняется по паре `(source, id)`.
- Пути и конфигурация должны управляться через переменные окружения и единый конфигурационный слой.
- Хардкод путей, секретов и переменных окружения запрещён.
